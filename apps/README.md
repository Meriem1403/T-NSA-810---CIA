# Application / site interne

Exemples acceptables pour le sujet « site web interne uniquement » :

- **Nginx** + pages statiques sur VM LAN S1.
- **Docker Compose** minimal (versionné ici) pour déploiement reproductible hors secrets.

Critère : jointure LAN/VLAN documentée dans `docs/architecture.md` ; exposition Internet directe absente ou bloquée en démo.
